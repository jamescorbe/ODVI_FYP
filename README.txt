install requirements before running 

cd {HOME}/ODVI_FYP
pip install -r requirements.txt

-----------------------------------------------------------
Running

Specify 0 for the primary system camera and 1 for the secondary,
any other input will use the test video for inference.

Passing True will enable video stream to view the inference 
results.

------------------------------------------------------------

for audio based tests!
